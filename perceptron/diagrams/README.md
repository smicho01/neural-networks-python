# Diagrams Directory

This directory contains all Mermaid diagrams used in the Perceptron documentation.

## Files

### 1. **perceptron-architecture** 
   - **Files**: `perceptron-architecture.mmd`, `perceptron-architecture.md`
   - **Description**: Shows the complete architecture of a perceptron with inputs, weights, summation, activation function, and output
   - **Best for**: Understanding the structure and components

### 2. **prediction-flow**
   - **Files**: `prediction-flow.mmd`, `prediction-flow.md`
   - **Description**: Step-by-step flowchart of how prediction works (forward propagation)
   - **Best for**: Understanding the prediction process

### 3. **training-algorithm**
   - **Files**: `training-algorithm.mmd`, `training-algorithm.md`
   - **Description**: Complete training algorithm flowchart showing epochs, error calculation, and weight updates
   - **Best for**: Understanding how the perceptron learns

### 4. **linear-separability**
   - **Files**: `linear-separability.mmd`, `linear-separability.md`
   - **Description**: Visual comparison of linearly separable vs non-linearly separable problems
   - **Best for**: Understanding perceptron limitations

## File Formats

- **`.mmd`**: Pure Mermaid syntax (for online editors like mermaid.live)
- **`.md`**: Markdown files with:
  - Mermaid diagrams in code blocks
  - ASCII art alternatives (for terminals/editors without Mermaid support)
  - Detailed descriptions and explanations

## Viewing Options

### Option 1: PyCharm
Open the `.md` files - they contain ASCII art versions that display properly in PyCharm.

### Option 2: Online Mermaid Editor
1. Visit: https://mermaid.live/
2. Copy content from `.mmd` files
3. Paste and view interactive diagrams

### Option 3: VS Code
Install "Markdown Preview Mermaid Support" extension to view Mermaid diagrams directly.

### Option 4: GitHub
GitHub automatically renders Mermaid diagrams in Markdown files.
