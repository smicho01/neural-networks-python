# Neural Networks - From Perceptron to GPT-3

A comprehensive educational project covering neural network fundamentals, from implementing a simple perceptron to understanding how GPT-3 works.

---

## What's Included

This repository contains two complementary components:

### 1. Structured Learning Course
**Location:** `/some_reading/`

A complete course taking you from zero to understanding modern AI:
- 7 lessons covering fundamentals to advanced concepts
- Beginner-friendly with clear explanations
- No prior ML knowledge required

[**Start the course →**](some_reading/README.md)

### 2. Hands-On Implementation
**Location:** `/perceptron/`

Working Python code implementing a perceptron:
- Complete implementation with detailed comments
- Examples: OR gate, AND gate, custom classifications
- Visual diagrams and explanations

[**See the code →**](perceptron/README.md)

### Recommended Approach

```
1. Read Lesson 1-2 (Introduction & Perceptron Basics)
```
